# Practical 9

## Magic Square

This program reads a file containing a square matrix of positive integers and checks if it is a magic square. A magic square is a matrix of integers where the rows, columns and diagonals sum to the same value. 

To compile and run this program enter the following command into the terminal:

```
gcc main.c -o main
./main
```

Or type `make` into the terminal to compile the `c` program through the `Makefile`, then type `./main` to run the the program executable.

## Output

* Entering the file name `magic_square.txt`:

```
Enter file name: magic_square.txt
No. lines, 3
This square is magic
```

* Entering the file name `not_magic_square.txt`:

```
Enter file name: not_magic_square.txt
No. lines, 3
This square is NOT magic
```
